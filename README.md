# City Signal Twin

A micro digital-twin project that fuses weather, transit delay, and air-quality signals to estimate commuter stress hotspots.

## Why this is portfolio-worthy
- Cross-domain data fusion with clear user value.
- Strong visualization narrative.
- Good base for geospatial and forecasting work.

## MVP scope
- Ingest weather + AQ + transit delay mock data
- Compute Stress Index by area
- Map view + hourly trend

## Tech stack
- Backend: Python
- Frontend map: MapLibre
- Data: Open APIs + synthetic fallback
